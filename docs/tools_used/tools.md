<h2>Tools</h2>

This page details the main tools, libraries, and infrastructure used in the project.

<strong>Main Libraries</strong>
- **Streamlit:** Web framework for building interactive dashboards quickly and Pythonically.
- **Plotly:** Interactive data visualization library, integrated with Streamlit for dynamic charts.
- **Pandas:** Powerful library for tabular data manipulation and analysis.
- **Jupyter Notebook:** Environment for initial data exploration, experimentation, and cleansing.

<strong>Testing tools</strong>
- **PyTest:** Framework for automated unit and integration testing of the application.
- **Selenium:** Tool for automated testing of the web interface, simulating user interactions.

<strong>Infrastructure</strong>
- **Terraform:** Infrastructure-as-code tool used to describe and provision cloud resources, ensuring reproducibility and scalability.

<strong>Cloud Platform</strong>
- **AWS EC2:** Cloud-based virtual machine hosting the Streamlit dashboard, providing continuous access to end-users.

<strong>CI/CD</strong>
- **GitHub Actions:** Continuous integration and deployment pipeline configured to automatically build, test, and deploy the project, ensuring code quality and delivery speed.

<strong>Version Control</strong>
- **Git & GitHub:** Source code versioning, team collaboration, and traceability of contributions through branches and pull requests.
